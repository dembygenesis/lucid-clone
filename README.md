## Quick Start

```bash
npm install
```

## Features

- Infinite canvas with shapes (rectangles, circles, diamonds, text)
- Real-time collaboration via WebSockets
- Connectors between shapes
- Auto-save to SQLite backend
- Export diagrams

## Tech Stack

- **Frontend:** React + TypeScript + Konva.js
- **Backend:** Node.js + Express + Socket.io
- **Database:** SQLite + Drizzle ORM
